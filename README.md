# rrai-desktop
软软 AI 桌面版程序

软软 AI 网站[https://rrai.idns.link](https://rrai.idns.link)
## 软软 AI 桌面版
微信小程序有一些限制，再加上一些工作还是在 PC 上完成，所以写一个桌面版。

### 功能
软软 AI 的一些功能
- 实现了调用 OpenAI 的接口实现智能的问题
- 调用AIGC 的接口实现图片的生成
- 可以交流提示词
- 以及会员、积分、一些小工具

未来实现更多的功能

### 技术栈

- Tauri 一个类似Electron的Rust语言实现的框架。
- React 
- Antd
- Sqlite
